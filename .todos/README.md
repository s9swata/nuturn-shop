# Task Logs

This directory contains timestamped task logs from AI assistant sessions.

## Log File Naming Convention

Files are named with ISO 8601 timestamp: `YYYY-MM-DD_HH-MM-SS.md`

## Log Format

Each log file should contain:

```markdown
# Task Log - [Date/Time]

## Session Overview
Brief description of what the session aimed to accomplish.

## Tasks

### [Task Name]
- **Status**: pending | in_progress | completed | cancelled
- **Priority**: high | medium | low
- **Description**: What needs to be done
- **Notes**: Any relevant notes or blockers

## Summary
- Completed: X tasks
- In Progress: Y tasks
- Pending: Z tasks
```

## Purpose

These logs help:
- Track progress across sessions
- Maintain continuity between AI assistant interactions
- Document decisions and implementation details
- Provide context for future work
